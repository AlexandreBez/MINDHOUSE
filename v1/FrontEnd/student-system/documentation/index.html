MINDHOUSE PROJECT

This project was generated with Angular CLI version 15.1.1.

GITHUB

The main project is located at Repository.

The Ideia

The main Ideia of this project is to create a selling service of courses using Spring boot Microservices, Angular, RabbitMQ and many other tools.

Front End

Development server

Run ng serve for a dev server. Navigate to http://localhost:4200/. The application will automatically reload if you change any of the source files.

Back End

Eureka Server

To check the serves visit Eureka Server.

RabbitMQ

To check Exchanges and Queues visit RabbitMQ Admin Page.
